# NUMBER-GUESSING-GAME
A random guessing game is a simple game where a player tries to guess a number (or item) that has been randomly chosen by the system.
A random guessing game is a simple game where a player tries to guess a number (or item) that has been randomly chosen by the system.


🎮 Basic Description:
In this game, the computer generates a random number within a given range (for example, 1 to 100). The player then tries to guess the number


🧠 How it Works:
The system selects a random number.
The player enters a guess.
The system gives feedback:
“Too high” if the guess is greater than the number
“Too low” if the guess is smaller
“Correct” if the guess matches the number
The game continues until the player guesses correctly.



🏆 Objective:
The goal is to guess the correct number in the least number of attempts.


🔁 Variations:
Limited number of attempts (like 5 or 10 tries)
Score based on number of attempts
Difficulty levels (easy: 1–50, hard: 1–1000)
Multiplayer mode


💡 Example:
Computer picks: 42
Player guesses: 60 → “Too high”
Player guesses: 30 → “Too low”
Player guesses: 42 → “Correct!”
